Technologies

The Big Three: Java, C++, and Javascript


I have used three programming languages extensively in my education and career so far:

  1. Java

    2. Java is where it all began. In high school I took an AP Computer Science course as my introduction to programming and picked up Java as my first language. Since then, I have used it extensively in several collegiate courses, namely Data Structures and Algorithms. It was in Java that I first implemented linked lists, stacks, queues, and learned about sorting. Since then, I have used Java to implement networking protocols with socket programming, and discover the asymptotic run time complexity of almost any given algorithm using the Master Theorem.

  2. C++

    3. I started working with C++ in the summer of 2013, creating a standalone graphical user interface for telecommunications software in a research lab at UTD. I used Qt4 to create a desktop application that allowed a user to easily manipulate configuration files. Through learning C++, I gained exposure to Unix operating systems, file manipulation, database management, and version control.

  3. Javascript

    4. Recently, I have begun using Javascript in several different ways.

    • I have used it to customize the behavior of a Customer Relationship Software called Netsuite, modifying customer and partner records, setting up user event field changes, and creating scripted reports and updates touching hundreds of thousands of transaction records.
    • I have employed the Node.js runtime environment to act as a middle man between Netsuite and Shopify, updating data as it changes in the form of a cron task.
    • Finally, I have created web applications and web sites such as this one with Node.js, Express, and Mongo to perform various tasks, including acting as a reporting system and bundle management system so ecommerce giant Shopify can behave as a custom inventory management system.

However, I firmly believe that programming languages are merely tools to solve critical thinking problems. It is not my grasp of C++, for example, that is valuable, it is my underlying understanding of Computer Science and problem solving that matters.

Low Level

The degree I am working towards is Computer Engineering, an intermixing of hardware and software, for example in the field of embedded systems. Through my coursework at UTD, I have become familiar with the Xilinx ISE design suite and MIPS assembly language in my digital design and computer architecture classes.

With a team of students, I designed and programmed an FPGA board to act as a basic CPU, complete with an Arithmetic Logic Unit and RAM.
